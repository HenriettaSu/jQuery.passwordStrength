# passwordStrength plug-in for jQuery

簡單的密碼強度檢測插件。

## DEMO

[示例](http://henrie.pursuitus.com/adminTemplate/index.html)


## 使用

```js
$(element).passwordStrength();
```

數字、字母、特殊字符、長度12位，每符合一種情況，強度上升一級。
單一字符重複，強度下降一級。
